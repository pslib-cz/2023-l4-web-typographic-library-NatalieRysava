    Headings

    Search engines use the headings to index the structure and content of your web pages. Users often skim a page by its headings. It is important to use headings to show the document structure. In the code sample below you can see how to implement heading in your code.

    Use <h1>,<h2>,<h3>,<h4>,<h5>,<h6> tags for headings.

    This is h1

    This is h2

    This is h3

    This is h4

    This is h5
    This is h6

    Optionally, class="underline" underlines the heading to separate it from the rest of the content.

    Styled text

    If you want to emphasize specific words in the text, you can use formatting effects such as bold using the <b> tag, or for greater importance <strong>. To format italic text, use <em> for emphasis and <i> for stylistic purposes. To highlight something, use the <mark> tag. Apply <a> to create a hypertext link and use <s> to visually strike out text with a line through it, but <del> for defining deleted text.

    Lists

    Ordered list

    The <ol> tag is used to create an ordered list. Combine it with the <li> tag to define the list items within the ordered list.

    1. Item 1
    2. Item 2
    3. Item 3 with two sub-items
      1. Sub-item 1
      2. Sub-item 2
    4. Item 4

    Unordered list

    The <ul> tag is used to create an unordered list. Combine it with the <li> tag to define the list items within the unordered list.

    • Item 1
    • Item 2
    • Item 3 with two sub-items
      • Sub-item 1
      • Sub-item 2
    • Item 4

    Buttons

    Button is created using class="button" and is primarily intended for light backgrounds. You can achieve a hover effect by adding class="button--hover".

    Button

    To use the button on a dark background, add "class: button--reversed".

    About project

    Glyphica.css is a free typographic library. It was created for as a school project. You don't need to make new CSS file every time you start new project. Glyphica.css helps you style your headings, special text markups, lists, basic tables, images and buttons. It's responsible and cros-browser friendly.
